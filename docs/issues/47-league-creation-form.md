# リーグ作成フォームの実装

**Issue**: #47
**URL**: https://github.com/aki05162525/kojiro-mahjong-app/issues/47

## 概要
リーグ一覧ページからリーグを新規作成できるフォームUIを実装する。

## 受け入れ条件
- [ ] リーグ作成ボタンがリーグ一覧ページに配置されている
- [ ] ボタンクリックでリーグ作成ダイアログが表示される
- [ ] フォームでリーグ名、説明を入力できる
- [ ] バリデーションエラーが適切に表示される
- [ ] 作成成功後、リーグ一覧が更新される
- [ ] 作成成功時に適切なフィードバック（トースト等）が表示される
