## GitHubへの登録方法

各issueファイルの内容を使って、GitHub上で個別にissueを作成してください。

### 手順：
1. GitHubリポジトリのIssuesタブを開く
2. 「New issue」ボタンをクリック
3. 各issueファイルの内容をコピーして貼り付け
4. タイトルには各ファイルの見出し（# の後の文字列）を使用
5. ラベルには各ファイルに記載されているラベルを設定
6. 「Submit new issue」をクリック

これを4つのissueそれぞれについて繰り返してください。
