# リーグ管理画面の実装

**Issue**: #49
**URL**: https://github.com/aki05162525/kojiro-mahjong-app/issues/49

## 概要
リーグ一覧ページとリーグ設定変更フォームを実装する。

## 受け入れ条件

### リーグ一覧ページ
- [ ] リーグ一覧が表示される（`/leagues`）
- [ ] リーグカードをクリックするとリーグ詳細ページに遷移する
- [ ] ステータス（進行中/完了）が表示される
- [ ] 作成日が表示される

### リーグ設定変更フォーム
- [ ] リーグ詳細ページから設定変更画面にアクセスできる
- [ ] リーグ名、説明を編集できる
- [ ] ステータスを変更できる
- [ ] バリデーションエラーが適切に表示される
- [ ] 更新成功後、詳細ページに戻る
- [ ] 更新成功時に適切なフィードバックが表示される
