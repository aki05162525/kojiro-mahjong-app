# AI Tips: デザインリファクタ

## Atlassian Design System を使ったリファクタ

Atlassian Design System のドキュメントを参照してデザインのリファクタを実施する。

**参考ドキュメント:**
- https://atlassian.design/components

**使い方:**
AIにドキュメントを読ませて、既存のコンポーネントを Atlassian Design System のベストプラクティスに沿ってリファクタさせる。

**例:**
```
Atlassian Design System (https://atlassian.design/components) のドキュメントを参照して、
app/components/Button.tsx をリファクタしてください。
```
